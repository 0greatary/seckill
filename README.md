# 秒杀插件(seckill)

浏览器 抢购、秒杀插件 可自定义 秒杀辅助插件（减少人肉失误）

* 任意网站，自定义添加秒杀定时任务
* 支持可视化选择“秒杀”按钮+自定义选择dom元素
* 自定义秒杀频率，秒杀次数
* 秒杀前2分钟提醒
* 北京时间自动校对，准时秒杀

#### 注意事项
* 请求提前登陆，选择商品的规格型号
* 暂不支持秒杀时输入验证码验证和其他复杂的（多步骤）抢购秒杀
* 秒杀助手只是辅助，不保证100%秒杀成功
